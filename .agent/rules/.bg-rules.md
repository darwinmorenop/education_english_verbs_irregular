# Irregular Verbs Project Rules

## Browser Usage
- **DO NOT** use the `browser_subagent` or any browser-related tools to verify results unless explicitly requested by the user. 
- Trust the source code analysis and build results unless a visual check is critical.

## Environments
- The project uses `environment.ts` for Production and `environment.development.ts` for Development.
- `ng serve --configuration development` (the default `npm start`) uses `environment.development.ts`.
- `exerciseCount` in `environment.development.ts` controls the number of questions in the Exercise view.
